# Sistem Rekomendasi Buku

Implementasi Information Retrieval menggunakan metode TF-IDF dan Cosine Similarity.

## Dataset
Books Dataset

## Metode
- TF-IDF
- Cosine Similarity
- Content-Based Filtering

## Cara Menjalankan
1. Buka notebook di Google Colab
2. Upload dataset
3. Jalankan seluruh cell
4. Masukkan judul buku
5. Sistem akan menampilkan rekomendasi buku yang mirip.
